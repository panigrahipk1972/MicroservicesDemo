Answer:

Two-Way Data Binding keeps the UI and component property synchronized. Angular implements it using [(ngModel)]. Any change in the input control updates the component property and any change in the component property automatically updates the UI.

Question
How do you call a Microservice from Angular?
Answer

Angular uses the HttpClient service from @angular/common/http to make asynchronous HTTP requests (AJAX calls) to backend APIs or microservices. Typically, we encapsulate the API calls inside Angular Services and consume them from Components using RxJS Observables and subscribe().

What is RouterLink?
RouterLink is a directive used for navigation between routes in an Angular application.

RouterLink is an Angular directive used to navigate between routes without reloading the entire page. It works with Angular Router to provide SPA (Single Page Application) navigation.

What is RouterOutlet?
RouterOutlet is a placeholder where Angular dynamically loads components based on the current route.

RouterOutlet is a directive that acts as a placeholder in the view where Angular dynamically renders the component associated with the active route.

If an interviewer asks:
"Explain RouterLink and RouterOutlet."
You can answer:

RouterLink is used to navigate between Angular routes, while RouterOutlet acts as a placeholder where the routed component is rendered. RouterLink is similar to ActionLink in ASP.NET MVC, and RouterOutlet is similar to RenderBody in a Razor Layout.

An empty path route acts as the default route of the application. When users access the root URL, Angular automatically redirects them to a designated page such as Home, Dashboard, Login, or Create Order.

How do you implement a 404 page in Angular?

Angular provides a wildcard route ('**') that matches any URL not handled by previous routes. We typically map it to a NotFoundComponent and place it as the last route in the routing configuration.

What is HttpClient?

HttpClient is an Angular service used to communicate with backend APIs using HTTP methods such as GET, POST, PUT and DELETE.

What does HttpClient return?

HttpClient returns an Observable.

How do we get data from an Observable?

By using the subscribe() method.

What is subscribe()?

subscribe() is used to receive data from an Observable. HttpClient methods return Observables, and subscribe() allows us to handle successful responses, errors, and completion events.

Explain HttpClient and subscribe in Angular."
You can now confidently explain:

HttpClient returns an Observable. An Observable is a stream that may produce data in the future. We use subscribe() to listen for that data. The next callback handles successful responses and the error callback handles failures such as network or server errors.

Services
Services can be used to share data between components. A singleton service stores shared state, and RxJS Subjects or BehaviorSubjects can be used to notify multiple components when data changes.

If asked:

What is the difference between a Service and a Factory?

Give this answer:

A Service is a reusable class containing business logic and is managed by Angular Dependency Injection. A Factory is a function responsible for creating and configuring an object before it is injected. Services are commonly used for application logic, while factories are used when instance creation requires custom logic or runtime configuration.

Because it pushes updates automatically to all subscribed components and supports reactive programming using RxJS.
